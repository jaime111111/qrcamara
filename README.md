Ommatidia is an open-source image recognition library that scans multiple 1D barcodes at once. It is based on zxing.


## Supported Formats

| 1D product | 1D industrial | 2D
| ---------- | ------------- | --------------
| UPC-A      | Code 39       | N/A
| UPC-E      | Code 93       | 
| EAN-8      | Code 128      | 
| EAN-13     | Codabar       | 
|            | ITF           |
|            | RSS-14        |
|            | RSS-Expanded  |
